# 微信邮件监控快捷命令

## 支持的命令

在微信中，您可以使用以下命令来主动查看邮件：

### 中文命令

| 命令 | 说明 |
|------|------|
| `今日邮件` | 查看今天的所有邮件摘要 |
| `查看邮件` | 同上 |
| `邮件摘要` | 同上 |
| `今天邮件` | 同上 |
| `今天的邮件` | 同上 |

### 英文命令

| 命令 | 说明 |
|------|------|
| `today's emails` | 查看今天的所有邮件摘要 |
| `emails today` | 同上 |
| `email digest` | 同上 |
| `check emails` | 同上 |

### 快捷命令

| 命令 | 说明 |
|------|------|
| `/emails` | 查看今天的邮件 |
| `/email` | 查看今天的邮件 |
| `📧` | 查看今天的邮件 |
| `📬` | 查看今天的邮件 |
| `📨` | 查看今天的邮件 |

### 日常用语

系统还能识别以下日常用语：

- `看看邮件`
- `邮件呢`
- `今天有邮件吗`
- `有新邮件吗`

## 使用示例

```
你: 今日邮件
Bot: 📧 **今日邮件摘要**
    📅 2026年4月28日
    📊 共 5 封邮件

    👤 **sender@example.com** (2封)
      1. 项目进度汇报
         项目已完成80%，预计下周...
      2. 会议邀请
         请参加明天下午3点的...

    👤 **newsletter@company.com** (2封)
      1. 本周产品更新
         新版本发布了以下功能...
      2. 优惠活动通知
         限时优惠，不要错过...

    👤 **boss@company.com** (1封)
      1. 紧急：客户报告
         请立即处理客户的投诉...

    ⭐ **重要邮件**
    • boss@company.com: 紧急：客户报告
```

## 智能识别

系统会智能识别包含以下关键词的邮件，并将其标记为重要：

**中文关键词：**
- 紧急
- 重要
- 需要处理
- 会议
- 截止
- 审批
- 通知
- 警告

**英文关键词：**
- urgent
- important
- action
- meeting
- deadline
- approval
- notice
- warning

## 配置要求

要使用此功能，请确保：

1. 邮件渠道已正确配置
2. 微信渠道已登录（`nanobot channels login weixin`）
3. 您的微信用户 ID 在允许列表中（如果配置了 `allow_from`）

## 自定义命令

如需添加更多自定义命令，可以编辑 `nanobot/channels/commands.py` 文件中的命令映射。

## 注意事项

- 命令响应是即时的，无需等待定时执行
- 手动查看的邮件数量限制为 50 封（可在代码中调整）
- 如果您的微信设置了安全限制，可能需要先将 Bot 加为好友
- 命令区分大小写不敏感，支持中英文混合输入
